What is

Web Development?

It's the process of coding a website, using 3 main languages: HTML5, CSS3, and JavaScript6. It can also mean designing the layout and style of the site. There are also people who design and code the interactive elements of a site, known as UI & UX. You can also be the manager of the development team. We will learn the basics of some of these concepts.

There are 2 main developers:

Back & Front end

Back end, aka server side, can use many different programming languages (Ruby, Java, Python, PHP, C#, and JavaScript to name some popular languages). This is the behind-the-scenes part of a website, and we won't discuss it that much.

We will be concerned with front end, aka client side. This is the part that the user sees and interacts with. It can be a broad set of disciplines: coding, layouts, designs, user interfaces, and user experiences.

We will start with...

HTML

It stands for HyperText Markup Language, and the current version is 5. It's the foundation or format behind essentially every site. It was developed in the 80s and it was first introduced in 1993.

It is probably the easiest part to learn.

Next on to...

CSS

It stands for Cascading Style Sheets, and the current version is 3. CSS has a wide array of abilities, but it is mainly the style of a site. It was developed in the 90s and it was first introduced in 1996.

It has more concepts than HTML, so it can be more difficult.

Lastly we have...

JavaScript

It doesn't have a fancy meaning, and the current version is 6 (released last month so we will use 5). JS is a true programming language, so it can control a lot of different things such as: user interactions, saving things in a form, animation, drawing, math, and so much more.

There are many concepts to explain, so JS is the most difficult. It can also do a ton of cool stuff!
